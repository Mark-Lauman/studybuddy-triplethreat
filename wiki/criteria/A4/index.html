Assignment 4: Version 2

For the assignment, the requirements are the same as for version 1, plus the following:

  • Include a test plan indicating what tests you have performed, and plan to perform, on the system. Part of this test plan will of course include the unit testing you have been doing from the start, plus all other testing you plan to do.
  • Include a release version of your project --- a version that any user could download and run on their system. This release version should not contain any source code, just the executable code and documentation needed to run it. You must include installation instructions, and please also include any appropriate warnings, e.g. if you have not tested your system thoroughly than say so!
  • A very detailed work assignment document that uses a spreadsheet to show all the features and work required by your project, and who is responsible for them. At the very least, every distinct software feature must be assigned to someone. An estimate of the required amount of work and completion date should also be included.
